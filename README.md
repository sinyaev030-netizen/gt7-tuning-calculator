## 📊 О проекте

Этот калькулятор помогает подбирать настройки для автомобилей в GT7. Вы можете изменять параметры подвески, LSD, аэродинамики и трансмиссии, и сразу видеть, как меняются характеристики машины.

### ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🚗 **565+ машин** | Полная база автомобилей из GT7 с реальными характеристиками |
| 🏗️ **Подвеска** | Высота, пружины, стабилизаторы, демпферы, частота |
| 🔧 **LSD** | Начальный момент, ускорение, торможение (перед/зад) |
| 🌬️ **Аэродинамика** | Прижимная сила перед/зад |
| ⚙️ **Трансмиссия** | 8 передач, финальная передача, макс скорость |
| 🛑 **Тормоза** | Баланс тормозов |
| 📈 **Визуализация** | Радарная диаграмма характеристик |
| 💡 **Рекомендации** | Автоматические советы по улучшению |
| 🔍 **Поиск** | Быстрый поиск машин по названию |
